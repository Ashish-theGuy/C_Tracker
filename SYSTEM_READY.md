## 🚀 How It Works

### When User Asks About a City:

1. **YOLO Detection**: The system analyzes the city's video using YOLOv8 person detection
2. **Live Analysis Display**: Shows sample frames with bounding boxes around detected people
3. **Crowd Analysis**: 
   - Current crowd count
   - Crowd level (High/Medium/Low)
   - Trend (Increasing/Decreasing/Stable)
4. **AI Recommendation**: Should visit or avoid
5. **Alternative Suggestions**: Other cities with less crowd

## 📱 Frontend Features

- **Simple Prompt Bar**: Type questions naturally
- **Live YOLO Display**: Shows detection frames with bounding boxes
- **Real-time Analysis**: Crowd density and trends
- **AI Recommendations**: Visit/avoid suggestions
- **Alternative Cities**: Better options with less crowd
